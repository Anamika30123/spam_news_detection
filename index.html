About This Project

This project uses machine learning to detect spam or fake news articles. The model was trained on a dataset of labeled news articles and uses natural language processing techniques to identify patterns associated with misinformation.

Technologies Used:
  • Python with scikit-learn for machine learning
  • NLTK for natural language processing
  • Flask for the backend API
  • Bootstrap and JavaScript for the frontend
